# GPT-SoVITS 中文錄音腳本

100 句覆蓋現代漢語常見聲韻調的語料 — 朗讀後自動切片即可餵 GPT-SoVITS v2 訓練。

## 錄音規格（必讀）

- WAV / 16-bit PCM / 單聲道
- 採樣率：錄 48kHz 後 down-sample 到 32kHz（直接錄 32k 也可，但 48→32 信噪比較好）
- 訊噪比 ≥ 25dB（在安靜室內，距麥 15-20cm，避開窗邊／空調出風口）
- Peak −12dB ~ −6dB（不削波）
- 每句前後留 0.5s 靜音（slicer 會靠這個切）
- 不戴口罩、嘴唇潤潤再錄、避免吞口水／喘氣／椅子摩擦

## 朗讀指引

- 平穩語氣、自然斷句，**不要刻意誇張或念稿感**
- 句末降調收尾，標點清楚停頓 0.3-0.5s
- 念錯就**停 1.5s 後從整句開頭重來**（後處理時砍掉錯版）
- 一次錄 10-20 句休息一下，喉嚨乾就喝水
- 錄音檔命名 `001.wav` ~ `100.wav` 對應下表

## 錄音流程建議（一次完成版）

```
01. 開錄音軟體（Audacity / Reaper / Logic 都可）
02. 設定：48kHz / 16-bit / mono / 線性 PCM
03. 從第 1 句開始連續念到第 100 句，整段一次錄完
04. 中間休息時不要關錄音，留空白即可（後面會切）
05. 結束後 export 一個大 WAV：corpus_raw_48k.wav
06. 後處理：
    a. ffmpeg -i corpus_raw_48k.wav -ar 32000 -ac 1 corpus_32k.wav
    b. 用 GPT-SoVITS 提供的 slicer.py 自動切片到 ./dataset/raw/
    c. 用 funasr 或 whisper 自動轉寫成 .list 檔
    d. 人工校對 .list 確保文字 == 音檔內容（這步很重要）
07. 把 ./dataset/ 整個拷貝到訓練機，看 sovits_training_setup.md 開訓練
```

## 替代方案：先錄少量試水

如果擔心一次錄 100 句疲勞，可以先錄 1-30 句 (~5 分鐘)，跑一次完整訓練流程驗證 pipeline，
再決定要不要補錄到 100 句。實測 5-10 分鐘已經能產生堪用的聲音克隆，差別在「字音細節飽和度」。
